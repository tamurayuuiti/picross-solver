# Picross Solver

ピクロス（ノノグラム）をブラウザ上で自動解答する Web アプリケーションです。

React + TypeScript + Vite を基盤として構築されており、ヒント入力・検証・自動解答・結果プレビューまでをブラウザ上で完結できます。

---

* TypeScript
* Vite

## 動作フロー

1. 行ヒント・列ヒントを入力または貼り付け
2. ヒントの形式やサイズ整合性を自動検証
3. ヒント数から盤面サイズを自動決定
4. ソルバーを実行
5. 論理推論（Humanistic Phase）を実施
6. 必要に応じてバックトラック探索を実施
7. 解答結果を盤面および全体プレビューに表示
8. 矛盾や入力エラーがある場合は該当箇所を表示

## 主な機能

* 大規模盤面対応（100×100以上を想定）
* ソルバー統計情報の表示

## 使用しているアルゴリズム（概要）

### 論理推論（Humanistic Phase）

各行・各列について可能な候補パターンを生成し、全候補で共通して確定できるセルを埋めていきます。

### バックトラック探索（Backtrack Phase）

論理推論のみでは確定できない場合、候補を仮定しながら深さ優先探索を行います。

探索中は列ヒントとの整合性を逐次検証し、矛盾が発生した分岐を早期に打ち切ります。

## 入力仕様

### ヒント入力

* 空白区切り
* カンマ区切り

のいずれにも対応しています。

例:

```text
5
1 1
3
```

```text
5
1,1
3
```

### 盤面サイズ

盤面サイズはヒントの行数・列数から自動決定されます。

## 開発メモ
